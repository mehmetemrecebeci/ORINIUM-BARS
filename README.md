# ORINIUM-BARS

72V Modular UGV Electronics Platform

## Overview

ORINIUM BARS is a modular 72V unmanned ground vehicle electronics platform consisting of:

- Power Board
- Motor Driver
- Signal Board

The platform is designed around a 72V LiFePO4 battery architecture and focuses on modularity, scalability and maintainability.

## System Overview

72V Battery
→ Power Board
→ Motor Driver
→ Signal Board
→ Jetson Orin Nano
